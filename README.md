# 🎯 Student Grade Calculator

## 📌 Project Overview
This project is a simple Python-based Student Grade Calculator that:
- Takes student name and marks as input
- Validates input (marks must be between 0–100)
- Assigns grades using if-elif-else logic
- Displays encouraging messages based on performance

---

## 🚀 Features
- Input validation using while loop
- Function-based grading system
- User-friendly output format
- Encouraging feedback messages

---

## 🛠️ Technologies Used
- Python 3

---

## 📊 Grading Criteria

| Marks Range | Grade |
|------------|------|
| 90 - 100   | A    |
| 80 - 89    | B    |
| 70 - 79    | C    |
| 60 - 69    | D    |
| 0 - 59     | F    |
